# SQL-Sales-Data-Cleaning-Project
End-to-end SQL data cleaning project transforming raw sales data by fixing inconsistencies, handling NULL values, and standardizing formats for accurate analysis.

# Project Overview
This project focuses on cleaning and transforming raw sales data using SQL to make it ready for analysis and reporting.

# Objective
- Improve data quality
- Handle missing values
- Standardize formats
- Ensure data accuracy

# Tools Used
- SQL

# Dataset
The dataset contains raw sales data with multiple data quality issues such as:
- Inconsistent date formats
- Missing (NULL) values
- Currency symbols in numeric fields
- Extra spaces and special characters in text fields
- Inconsistent order status values
